# Stone Paper Scissors Game

A simple, console-based implementation of the classic Rock-Paper-Scissors game written in Python. This project provides a lightweight, well-documented example for beginners learning Python, game logic, and basic I/O.

- Clean, easy-to-understand codebase suitable for learning and extension.

## Requirements
- Python 3.8 or newer

## Installation
1. Clone or download this repository.
2. (Optional) Create and activate a virtual environment:

```bash
python -m venv venv
venv\Scripts\activate
```

3. No external dependencies are required for the base game.

## Usage
1. Open a terminal and navigate to the project directory.
2. Run the game:

```bash
python main.py
```

3. Follow the on-screen prompts to play.

## Gameplay
- The game asks the player for their choice: Rock, Paper, or Scissors.
- The computer (or second player) selects a choice.
- The winner is determined by the classic rules:
	- Rock beats Scissors
	- Scissors beats Paper
	- Paper beats Rock

## Project Structure
- `main.py` — Program entry point and game loop.
- `utils.py` — Helper functions used by the game.
- `README.md` — Project documentation.
